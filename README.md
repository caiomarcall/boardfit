# Projeto BoardFit

Este é um projeto acadêmico desenvolvido para criar uma interface funcional de uma landing page. O objetivo do BoardFit é ajudar surfistas a encontrarem a prancha ideal por meio de um Shaper Virtual baseado em um algoritmo simulado[cite: 1].

## Sobre o Projeto

A landing page utiliza técnicas de storytelling para guiar o visitante através de um problema real: a frustração de errar na escolha da prancha de surf[cite: 1]. A partir dessa dor, o BoardFit é apresentado como a solução ideal[cite: 1]. O sistema demonstra como calcular a litragem, definir as dimensões corretas e indicar as quilhas adequadas para a evolução do surfista, conectando o usuário a opções de mercado com foco em custo-benefício[cite: 1].

## Tecnologias Utilizadas

* **HTML5:** Utilizado para a marcação do conteúdo e estruturação das seções da página[cite: 1].
* **CSS3:** Utilizado para a estilização visual, definição da paleta de cores, layout dos cards e responsividade para dispositivos móveis[cite: 2].

## Estrutura de Arquivos

O projeto é composto por dois arquivos simples que devem ficar no mesmo diretório:

* **boardfit.html:** Contém toda a estrutura de tags e o conteúdo descritivo da landing page[cite: 1].
* **style.css:** Contém as regras de design, variáveis de cores e configurações de layout[cite: 2].

## Como Executar o Projeto

1. Baixe os arquivos `boardfit.html` e `style.css` e coloque-os dentro da mesma pasta no seu computador[cite: 1, 2].
2. Abra o VS Code e use a opção "Open Folder" para abrir essa pasta.
3. Clique com o botão direito no arquivo `boardfit.html` e selecione "Open with Live Server" para visualizar o site rodando no navegador.
