Projeto BoardFit 

Este é um projeto acadêmico desenvolvido para criar uma interface funcional de uma landing page. O objetivo do BoardFit é ajudar surfistas a encontrarem a prancha ideal por meio de um Shaper Virtual baseado em um algoritmo simulado.

Sobre o Projeto

A página foi reestruturada para aplicar técnicas de storytelling. O texto guia o visitante através de um problema real (a frustração de errar na escolha da prancha de surf) e apresenta o BoardFit como a solução ideal. A ferramenta demonstra como calcular a litragem, dimensões e indicar as quilhas corretas, sempre buscando opções de mercado com foco em custo-benefício.

 Tecnologias Utilizadas

* **HTML5:** Utilizado para a marcação do conteúdo e estruturação das seções da página.
* **CSS3:** Utilizado para a estilização visual, definição da paleta de cores, layout dos cards e responsividade para dispositivos móveis.

 Estrutura de Arquivos

O projeto é composto por dois arquivos simples que devem ficar no mesmo diretório:

* **boardfit.html:** Contém toda a estrutura de tags da página e o texto do storytelling integrado.
* **style.css:** Contém as regras de design, variáveis de cores e configurações de layout.

 Como Executar o Projeto

1. Baixe os arquivos `boardfit.html` e `style.css` e coloque-os dentro da mesma pasta no seu computador.
2. Abra o VS Code e use a opção "Open Folder" para abrir essa pasta.
3. Clique com o botão direito no arquivo `boardfit.html` e selecione "Open with Live Server" para visualizar o site rodando no navegador.
